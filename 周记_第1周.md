# 实训周记 — 第一周

**项目**：多用户多级目录文件系统（图形用户接口）  
**日期**：2026年6月 第一周

---

## 本周完成工作

### 1. 需求分析与调研
- 研读课程设计文档，梳理出 12 项核心命令需求：login、系统说明、create、open、read、write、close、delete、mkdir、cd、dir、logout
- 上网搜索类似开源项目，重点研究了 **sensnow/os-system-exp**（华南农业大学，JavaFX）和 **BangBOOM/File-System**（东北大学，Python）
- 对比各项目的架构设计，确定技术选型方向

### 2. 技术选型与环境搭建
- 确认开发环境：**JDK 17 + Swing**（无额外依赖）
- 设计四层架构：磁盘模拟层 → 文件系统核心层 → 用户管理层 → GUI 表示层
- 编写《实现计划》，规划 17 个源文件、7 步开发顺序

### 3. 系统架构设计
- **磁盘层**：128 块 × 64 字节 = 8KB，FAT 显式链接，目录项 16 字节
- **文件系统层**：树形多级目录，路径解析器，打开文件管理
- **用户层**：SHA-256 密码哈希，用户主目录 /home/ 隔离
- **GUI 层**：Swing（JTree + JTable + MenuBar + ToolBar + CommandLine）

### 4. 完成磁盘模拟层编码
- `DiskConstants.java`：系统常量定义
- `Disk.java`：byte[128][64] 内存数组 + disk.img 持久化
- `FAT.java`：块分配、释放、链遍历、空闲统计
